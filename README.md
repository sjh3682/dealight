# Dealight — 중고거래 가격 비교 · 적정가 판단 도구

물건 이름과 상태를 입력하면, 이미 올라와 있는 매물과 가격을 보여주고
**지금 시세가 얼마인지**, 그리고 내가 생각한 가격이 **비싼지/싼지** 알려주는 웹 사이트입니다.

- 디자인: 몬드리안(데스테일) 스타일 — 빨강·파랑·노랑 블록 + 굵은 검은 격자
- 기술: Python + Flask (서버), HTML/CSS (화면)
- 데이터: 실제 시세를 참고해 만든 **가상 매물 449건** (`data/listings.csv`)
  실제 사이트를 크롤링하지 않으므로 약관·법적 문제가 없습니다.
  나중에 직접 수집한 데이터로 CSV만 교체하면 그대로 작동합니다.

## 폴더 구성
```
dealight/
├── app.py              # Flask 서버 + 검색/시세 분석 로직
├── generate_data.py    # 가상 매물 데이터 생성기
├── data/
│   └── listings.csv    # 매물 데이터 (generate_data.py 로 생성됨)
├── templates/
│   ├── base.html       # 공통 레이아웃(헤더/푸터)
│   ├── index.html      # 검색 화면
│   └── results.html    # 결과(시세·진단) 화면
└── static/
    └── style.css       # 몬드리안 스타일 디자인
```

## 실행 방법
```bash
# 1) Flask 설치 (최초 1회)
pip install flask

# 2) (선택) 데이터 다시 만들기
python generate_data.py

# 3) 서버 실행
python app.py

# 4) 브라우저에서 접속
#    http://127.0.0.1:5000
```

## 사용 방법
1. 검색창에 물건 이름을 입력합니다. (예: `아이폰 14`, `에어팟 프로 2`)
2. 5단계 상태 중 하나를 고릅니다.
   (미사용 / 사용감 거의 없음 / 눈에 띄는 흔적 없음 / 사용감 있음 / 사용감 심함)
3. (선택) 생각 중인 가격을 입력하면 비싼지/싼지 진단해 줍니다.
4. **시세 확인하기**를 누르면 추천 시세·최저·평균·최고·적정 가격대와
   현재 올라온 매물 목록을 보여줍니다.

## 시세는 어떻게 계산하나요?
상태가 섞인 매물을 공정하게 비교하기 위해, 각 매물 가격을 기준 상태
('눈에 띄는 흔적 없음')로 환산한 뒤 중앙값을 구하고, 사용자가 고른 상태의
가격 배수를 다시 곱해 **추천 시세**를 만듭니다. 적정 가격대는 환산값의
25~75 백분위 구간을 사용합니다. (자세한 로직은 `app.py` 의 `analyze()` 참고)
